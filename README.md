## Viewpoints

- Security
- Privacy
- Availability
- Mobile Accessibility
- Maintainability
- Operation in a Remote Area

## Requirements, Needs, and Desires

| Code | Requirement, Need, or Desire |
| --- | --- |
| RND1 | The aqueduct employees must be able to monitor water tanks through a dashboard accessible from any mobile device |
| RND2 | The aqueduct employees must be alerted of high overflow or undersupply risk |
| RND3 | The aqueduct employees must be assured that their data cannot be externally modified, deleted, or accessed by unauthorized parties |
| RDN4 | The system must obtain water tank monitoring data from the exiting Octave ultrasonic meter and the 24V One-Wire depth sensor |
| P1 | The system must obtain its energy from a solar panel |
| P2 | The system must have a battery with an autonomy of at least 3 months |
| M1 | The system must be contained in a cabinet that can withstand environmental conditions |
| M2 | The system should operate for at least 48 continuous hours under direct exposure to rain or sunlight |
| C1 | The system must implement mobile connectivity infrastructure |
| C2 | The system must avoid mobile connectibity interruptions |
| CS1 | The system must be cybersecurity-hardened following applicable standards |
